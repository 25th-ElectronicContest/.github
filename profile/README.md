# 2025届电赛

## 描述

此仓库为2025届电赛仓库，用于存放硬件和软件代码，此仓库目前将设为私有仓库

## 仓库结构

~~~ bash
Hardware		# 硬件文件存储仓库
Software		# 软件代码存储仓库
Vision      # 上位机视觉代码存储仓库
~~~

目前仓库均为私有，将在后续完成作品后再公开
